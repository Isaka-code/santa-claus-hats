# santa-claus-hats


## ディレクトリ構成

```
santa-claus-hats/
├── app.py                 # Streamlitアプリのメインスクリプト
├── requirements.txt       # 必要なパッケージ一覧
├── assets/                # 静的ファイルを格納
│   └── santa_hat.png      # サンタクロースの帽子画像
├── outputs/               # ユーザーが生成した画像の保存先
└── utils/                 # ユーティリティ関数やヘルパースクリプトを格納
```
